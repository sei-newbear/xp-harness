# E2E の実行手順

E2E テストを実行する手順 (実行環境のセットアップ、実行コマンド、CI 統合、実行失敗時の対処) を扱うスキル。手順そのものはプロジェクト側にあり、このスキルはそれを探して効かせる入口。spec の書き方は別スキルの責務。

## 関連する E2E の実行手順を探して従う

このスキルが呼ばれるたびに、以下を行う（1 つの作業の中で触る範囲が別の領域に移ったら、その都度やり直す）:

1. **触る範囲を特定する**: いま動かそうとしている E2E のディレクトリ / 領域を特定する（例: front 配下と api 配下で実行手順が分かれている場合、どちらの領域の作業かを特定する）
2. **範囲に対応する実行手順を探す**:
   - セッションに登録されているスキル（このスキル自身を除く）にその範囲の E2E 実行手順を扱うものがあれば、（参照して読むのではなく）スキルとして呼ぶ
   - スキルでない手順ファイル（その領域のディレクトリ配下の CLAUDE.md / AGENTS.md 等のドキュメント）があれば、読む
3. **見つけた実行手順に従って動かす**
4. **見つからなければ、手順をでっち上げない**: 一般に良いとされる判断で進める
