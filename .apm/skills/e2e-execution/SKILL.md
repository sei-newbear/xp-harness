# E2E の実行手順

E2E テストを実行する手順 (実行環境のセットアップ、実行コマンド、CI 統合、実行失敗時の対処) を扱うスキル。spec の書き方は別スキルの責務で、ここは spec を「動かす」側を扱う。

## 関連する実行手順を探して従う

E2E の実行手順をまとめたスキルやルールがプロジェクトにあれば、探して読み、従って E2E を動かす。
